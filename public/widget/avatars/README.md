# Widget Avatarları

Bu klasöre **8 adet** avatar görseli yerleştirin. Widget bunları otomatik olarak kullanır.

## Beklenen Dosya Adları

```
avatar-1.svg   (veya .png)
avatar-2.svg
avatar-3.svg
avatar-4.svg
avatar-5.svg
avatar-6.svg
avatar-7.svg
avatar-8.svg
default.svg    ← avatar_id null olduğunda gösterilir
```

## Önerilen Boyut

- **40×40 px** veya vektör (SVG tercih edilir — hafif ve keskin)
- Yuvarlak görünüm CSS ile sağlanır (`border-radius: 50%`), görsellerin kare olması yeterli.

## Ücretsiz Kaynak Önerileri

- https://www.dicebear.com — programatik SVG avatar üreteci
- https://pravatar.cc — fotoğraf tarzı placeholder avatar
- Kendi illüstrasyon setin
